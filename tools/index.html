Personal Access Token
🔒
Your token is used only in this app to call the GitHub API. It is never transmitted to any external server — only directly to api.github.com. Revoke anytime at GitHub → Settings → Developer Settings → Personal Access Tokens.
Need a token? Create one on GitHub ↗  ·  Required scope: repo
How to create a token
01
Go to github.com → Settings → Developer Settings → Personal Access Tokens → Tokens (classic)
02
Click Generate new token (classic). Name it "Urban Trend". Expiry: 90 days.
03
Tick only the repo scope. Click Generate, copy the token, paste it above.
